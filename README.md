# dotfiles

自身が利用する環境ファイルをこちらで管理する。

## Getting started

### 1. Install tools

必要なツール等をインストールする。

■ Linux

```sh
sudo apt install stow
```

■ macOS

```sh
brew install stow
```

### 2. Run setup.sh

自身のOSに従って任意のシェルスクリプトを実行する。

■ Linuxの場合

```sh
cd linux
chmod +x setup.sh
./setup.sh
```
